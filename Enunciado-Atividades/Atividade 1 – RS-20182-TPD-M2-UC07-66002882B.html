          Atividade 1

          Contexto:

          Em uma reunião entre os gestores da XYZ Game Studio, foi decidido realizar o lançamento do Alien City para a plataforma mobile. As equipes foram realocadas para adaptar as jogabilidades às fases, ou seja, adequar os gráficos e controles para mobile. A sua equipe ficou responsável pela fase 5. Logo, seu gestor solicitou o desenvolvimento de um protótipo para implementar uma palanca analógica para a plataforma Android. Em seguida, pediu que programasse esse controle para a fase 5, da caverna do Alien City e que planejasse a vibração para os casos em que houver dano, ou morte.

          Atividades:

          Para realizar as adaptações solicitadas, você deverá seguir os seguintes passos:

          1.º – Dentro de Materiais Complementares, fazer o download do Prototipo da Fase 5 do Alien City que contém assets de personagem, cenários e obstáculos da caverna.

          2º - No protótipo, crie a programação dos controles para mobile. Implemente os comandos de joystick para controlar o personagem. Para isso, abra a cena, inclua o asset de joystic virtual da asset store e programe o script de touch para movimentar o personagem com a palanca.      

          3º Utilize o GitHub e, no repositório, crie uma Branch “feature” para adaptar ao mobile.

          Entrega           

          Crie uma pasta com o nome <SeuNome_UC7_A1_AlienCity_Mobile>.

          Dentro da pasta acima copie as pastas Assets e Project Settings de seu Projeto

          Acrescente na pasta acima um relatório com histórico de revisões, obtido com um print da tela, pelo Github Desktop na visão History do repositório, ou pelo Github Shell com o comando git log.

          Compacte a pasta criada em arquivo de mesmo nome e entregue-a no AVA, no local destinado para a tarefa, até a data informada no cronograma.

          Avaliação

          Nesta atividade, você deverá evidenciar os seguintes indicadores:

          • Implementa e valida protótipo para subsidiar o desenvolvimento otimizado do código do jogo digital para dispositivos móveis.
          • Implementa a interação dos elementos multimídia do jogo digital para dispositivos móveis a partir da validação do protótipo.
          • Depura, documenta, otimiza e aperfeiçoa o código de desenvolvimento para jogo digital em dispositivos móveis.
          • Realiza os procedimentos de versionamento por meio do repositório conforme as normas de segurança de dados.
